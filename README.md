# would-you-rather
My Project-2 of Udacity Nanodegree

Installation:

1. Clone this repo using `git clone https://github.com/KirankumarAmbati/would-you-rather.git`
2. `cd would-you-tather`
3. `npm install`
4. `npm start`
5. Hit `http://localhost:3000/` in browser to view the output.
